## <a name="more-information-about-bots"></a>Weitere Informationen zu Bots

Wenn Sie mehr über die Verwendung dieses Diensts mit Bots erfahren möchten, beginnen Sie mit den folgenden Ressourcen:

|Resource|Zweck|
|--|--|
|[Azure Bot Service](https://dev.botframework.com/)|Azure Bot Service bietet einen vollständigen in der Cloud gehosteten Webdienst mit einem Bot-Endpunkt. Der Dienst verwendet das [Bot Framework](https://github.com/Microsoft/botframework), das in mehreren Sprachen verfügbar ist.|
|[Bot Framework](https://github.com/Microsoft/botframework)|Microsoft Bot Framework ist eine umfassende Plattform für den Aufbau unternehmensweiter, dialogorientierter KI-Erfahrungen.|
|[Bot Framework-Emulator](https://github.com/Microsoft/botframework#Bot-Framework-Emulator)|Bot Framework Emulator ist eine plattformübergreifende Desktopanwendung, mit der Entwickler von Bots mit dem Bot Framework SDK erstellte Bots testen und debuggen können. Sie können Bot Framework Emulator verwenden, um lokal auf Ihrem Computer ausgeführte Bots zu testen oder eine Remoteverbindung mit ausgeführten Bots herzustellen.|
|[Bot Framework-Tools](https://github.com/Microsoft/botbuilder-tools)|Die Bot Framework-Tools sind eine Sammlung plattformübergreifender Befehlszeilentools, die entwickelt wurden, um den gesamten Workflow bei der Bot-Entwicklung abzudecken. |
|[Beispiele für das Erstellen von Bots](https://github.com/Microsoft/BotBuilder-Samples)|Vollständig entwickelte Beispielbots sind darauf ausgelegt, Szenarien zu veranschaulichen, die Sie implementieren müssen, um großartige Bots zu erstellen.|
