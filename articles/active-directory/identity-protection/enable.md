# <a name="enabling-azure-active-directory-identity-protection"></a>Aktivieren von Azure Active Directory Identity Protection

- Eine einheitliche Ansicht gekennzeichneter Benutzer und Risikoerkennungen, die mithilfe eines Algorithmus f체r maschinelles Lernen ermittelt wurden
